# Relatorio_Faturamento_Usinas
  Criação de relatório em PowerBI do faturamento de usinas de 2020 a 2021 utilizando pandas para tratamento de dados e criação da base para o relatório. 
  Os relatórios de receita são públicos e estão disponíveis em:
  
  https://www.ccee.org.br/web/guest/mercado/leilao-mercado --> Links de Apoio --> Relatórios de Receita
  
  Os nomes das usinas e o CEG podem ser obtidos em: 
  
  https://www.ccee.org.br/web/guest/dados-e-analises/dados-geracao --> Download de Dados
